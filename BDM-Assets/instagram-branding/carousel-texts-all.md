## 1. 타르트 브랜드 트립 — "불공정이 보이면 알고리즘이 처벌한다"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
화장품 브랜드가 인플루언서 50명을
두바이 5성급 호텔에 초대했다.
수백만 달러를 쓴 이 행사가
브랜드를 망쳤다.

**[2] 현상**
문제는 단순했다.
A는 스위트룸 + 에르메스 가방
B는 일반 객실 + 기본 키트
둘 다 실시간으로 스토리에 올렸다.

**[3] 비교가 보이는 순간**
팔로워들은 두 사람의 스토리를
동시에 봤다.
같은 행사. 다른 대우.
"불공정하다"는 감정이 폭발했다.

**[4] 원숭이 실험**
카푸친 원숭이 실험 (Frans de Waal, 2003)
두 원숭이에게 같은 과제를 시킨다.
한 마리에겐 포도, 한 마리에겐 오이.
오이 받은 원숭이는 오이를 내던졌다.
→ 불공정 반응은 본능이다.

**[5] 알고리즘이 분노를 증폭한다**
사람들이 분노하면 댓글을 단다.
길고, 구체적인 비판 댓글.
알고리즘은 감정의 방향(긍정/부정)을 모른다.
댓글이 많으면 = 좋은 콘텐츠로 판단.
→ 분노가 도달을 폭발시켰다.

**[6] 제3자 비교 영상**
가장 많이 조회된 건
참가자의 게시물이 아니었다.
"A는 이걸 받았는데 B는 이것만"
→ 제3자가 만든 비교 영상이
수천만 조회를 기록했다.

**[7] 소셜 미디어의 불문율**
① 숨겨진 차등은 발각되면 더 크게 터진다
② 팔로워 수 ≠ 인간 가치 (사람들은 이걸 안다)
③ 과시는 임계점을 넘으면 동경 → 반감

**[8] 브랜드가 배울 것**
→ 차등 대우를 하되, 보이지 않게 하라
→ "소셜 미디어에서 보이는 것은 전부"라고 가정하라
→ 알고리즘은 분노를 먹고 자란다.
   공정성 위반은 최고의 알고리즘 연료다.

**[9] CTA**
소셜 미디어에는 보이지 않는 공정성 규칙이 있다.
그걸 어기면 알고리즘이 처벌한다.

## 2. 라이언에어 톤 — "고객을 놀리는데 팔로워가 느는 구조"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
이 항공사는
고객 불만에 공감하지 않는다.
"좌석이 좁아요" → "그래서 싼 거잖아"
팔로워 200만. 업계 engagement 1위.

**[2] 보통 브랜드의 문제**
대부분의 브랜드 계정:
친절하고, 정중하고, 무난하다.
→ 아무도 기억하지 못한다.
피드에서 어떤 브랜드인지 구별이 안 된다.

**[3] 라이언에어는 다르다**
비행기에 큰 눈 스티커를 붙인다.
자사 서비스의 단점을 희화화한다.
고객 불만을 조롱 톤으로 답한다.
이걸 매일, 일관되게, 절대 안 바꾼다.

**[4] 왜 이게 먹히는가: 유사사회적 관계**
TV 속 인물에게 친밀감을 느끼는 것처럼
사람들은 '캐릭터가 있는 계정'에
일방적 친밀감을 형성한다.
→ "이번엔 뭐라고 할까?" 기대하며 재방문

**[5] 알고리즘이 보상하는 것**
① 프로필 재방문 → 최강 engagement 신호
② 댓글이 '대화'가 된다 → 깊은 상호작용
③ 낮은 제작 비용 → 높은 ROMI
라이언에어는 텍스트+눈알 이미지만으로
항공업계 최고 engagement를 찍는다.

**[6] 핵심 조건**
아무나 무례하면 되는 게 아니다.
라이언에어는 "유럽 최저가 항공"이라는
포지셔닝과 톤이 완벽히 일치한다.
→ 톤은 포지셔닝에서 파생되어야 한다.

**[7] 데이터**
라이언에어 TikTok engagement rate: 4-6%
항공업계 평균: 0.5-1%
→ 5~10배 차이. 광고비 거의 제로.

**[8] 실무 프레임워크**
① 톤을 정하면 최소 6개월 절대 안 바꾼다
② 무난함 = 최악의 전략
③ 톤이 맞으면 제작 비용은 안 들어도 된다
④ 호불호가 갈려도 반응이 있으면 알고리즘이 민다

**[9] CTA**
알고리즘은 "좋은 콘텐츠"가 아니라
"반응이 있는 콘텐츠"를 보상한다.
당신의 브랜드에 캐릭터가 있는가?

## 3. Very Demure 밈 — "빈칸만 채우면 바이럴"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
한 여자가 출근 메이크업을 하면서
"Very demure, very mindful"이라고 말했다.
6단어.
TikTok 조회수 수십억.

**[2] 진짜 폭발은 그 다음**
요리사: "Very demure, very seasoned"
변호사: "Very demure, very litigious"
NASA: "Very demure, very cosmic"
수십만 명이 같은 구조로 영상을 만들었다.

**[3] 왜 이게 터졌을까**
이건 "좋은 콘텐츠"가 아니었다.
"따라 하기 쉬운 콘텐츠"였다.
→ "Very __, very __"
구조는 고정. 빈칸만 채우면 됨.
특별한 편집 기술 필요 없음.

**[4] 템플릿의 힘**
보는 것 = 수동적 (조회수만 올림)
따라 하는 것 = 능동적 (새 콘텐츠 생성)
→ 참여자가 곧 홍보자가 된다.
한 명이 만들면 그 팔로워에게 노출.
→ 자기 강화 루프 형성.

**[5] 관심사 침투**
같은 템플릿이지만 맥락이 다르다.
요리/법률/과학/패션/육아...
→ 하나의 밈이 수십 개 관심사에
동시에 침투한다.
알고리즘은 관심사별로 각 버전을 추천.

**[6] 문화적 순간이 되면**
참여 안 하면 "뒤처진다"는 느낌.
→ 밴드왜건 효과
→ 참여 자체가 소셜 화폐가 됨

**[7] 원본 크리에이터는?**
줄리안나 레베카: 수백만 팔로워 획득.
하지만 다음 바이럴은? 거의 불가능.
→ 밈의 주의는 광범위하지만 얕다.
= 크리에이터 경제의 구조적 불안정성.

**[8] 브랜드 액션**
① 콘텐츠를 만들 때:
  "이걸 어떻게 따라 할 수 있는가?"를 먼저 설계
② 템플릿은 단순할수록 좋다
③ 밈을 만드는 것 < 밈에 올라타는 것
  타이밍: 발생 후 3~7일이 골든타임

**[9] CTA**
완성도 높은 콘텐츠보다
따라 하기 쉬운 콘텐츠가 이긴다.

## 4. 우는 말 인형 — "불량품이 20,000개 팔린 구조"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
중국의 한 공장에서
말 인형의 입을 거꾸로 꿰맸다.
다음 날 주문량:
400개 → 20,000개.

**[2] 무슨 일이 있었나**
말의 해를 맞아 빨간 말 인형을 만들고 있었다.
웃는 입을 달아야 하는데
한 작업자가 입꼬리를 반대로 꿰맸다.
말이 울고 있었다. 🐴

**[3] 고객의 선택**
항저우의 한 고객이 이 불량품을 받았다.
환불? 거절.
대신 사진을 찍어 올렸다.
48시간 뒤, 공장은 생산 라인을
2개 → 10개 이상으로 확대했다.

**[4] 왜 결함이 더 잘 팔리는가**
예일대 연구 (Journal of Consumer Research, 2018):
→ 실수로 만들어진 제품을 더 선호한다.
이유: 실수는 희소하다 → 더 특별하게 느낀다.
단, 조건: 실수가 유일하고 비의도적이어야.

**[5] 감정 연결**
'牛马(뉴마)' = 소와 말 = 과로의 은어
"이게 출근할 때 내 표정이고,
웃는 말은 퇴근할 때 내 표정"
→ 우는 말은 제품이 아니라
감정 표현의 매체가 됐다.

**[6] 알고리즘 루프**
① 멈춤: 설 인형이 울고 있다? → 인지적 불일치
② 감정: "이게 나야" → 공감 + 유머
③ 공유: 놀라움 + 유대감 → 소셜 화폐
→ 세 단계가 동시에 작동하면 폭발.

**[7] 아무도 검색하지 않았다**
이 인형을 검색해서 산 사람은 거의 없다.
더우인(중국 TikTok) 피드에서 우연히 발견했다.
→ 발견형 커머스:
필요를 인식하기 전에 필요를 만들어주는 구조.

**[8] 기업이 배울 것**
① 결함 = 위기가 아니라 신호로 읽어라
② 기능적 우수성은 알고리즘 연료가 안 된다.
  감정적 공명("이게 나야")이 알고리즘 연료다.
③ 48시간 안에 생산 라인 전환한 속도가 핵심.

**[9] CTA**
알고리즘은 완벽함을 보상하지 않는다.
감정을 보상한다.

## 5. 뉴터 버터 (패턴 인터럽트) — "이상한 콘텐츠가 먹히는 구조"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
미국의 한 땅콩 버터 브랜드가
의미불명의 영상을 올렸다.
제품 소개도, 레시피도 아닌
그냥... 이상한 영상.
수천만 조회.

**[2] 뉴터 버터의 TikTok**
글리치 효과.
의미불명의 땅콩 캐릭터.
기존 브랜드 문법을 완전히 벗어남.
6개월 만에 팔로워 100만+ 증가.
공유율: 일반 브랜드의 4~7배.

**[3] 왜 멈추는가: 패턴 인터럽트**
뇌는 다음에 뭐가 올지 예측한다.
뷰티, 음식, 댄스... 피드는 예측 가능.
→ 예측이 틀리면 주의가 강제 환기.
= 스크롤이 멈춘다.

**[4] 알고리즘의 해석**
멈춤 = 체류 시간 증가 ✓
이해 안 됨 → 다시 봄 = 재시청 ✓
"이거 봐" → 공유 = 최강 신호 ✓
알고리즘은 "왜 멈췄는지" 모른다.
멈추기만 하면 된다.

**[5] 공유의 비밀: 신뢰 비용 제로**
보통 공유할 때:
"이걸 공유하면 나 어떻게 보일까?"
→ 이상한 콘텐츠는 이 비용이 없다.
"이거 봐, 이상하지 않아?"
= 지지가 아니라 발견 보고.

**[6] 이상함에도 규칙이 있다**
뉴터 버터의 콘텐츠는 무작위가 아니다.
① 땅콩 캐릭터 = 고정
② 특정 색감 = 고정
③ 서사만 변주
→ 재즈처럼: 코드 진행은 고정,
   즉흥 연주만 바뀐다.

**[7] 조건**
이 전략이 되려면:
✓ 저관여 브랜드 (FMCG 등)
✓ 인지도 자체가 핵심 자산인 카테고리
✗ 신뢰가 핵심인 브랜드 (금융, 의료)엔 위험

**[8] 실무 액션**
① '첫 0.5초'를 설계하라
  → 즉시 카테고리 판별되면 실패
② 공유의 심리적 비용을 낮춰라
  → "발견자" 역할을 부여
③ 이상함에 브랜드 코드를 입혀라
  → 규칙 없는 이상함 = 소음

**[9] CTA**
5,000~10,000개의 콘텐츠가
하루에 당신의 눈을 스친다.
기억에 남는 건 예측을 깬 것뿐.

## 6. 스탠리 텀블러 — "설명 말고 증거"
**Type B (현상/사례) · 1 캐러셀 · 9장**

**[1] 훅**
한 여성의 차가 전소됐다.
차 안에 있던 텀블러는 멀쩡했다.
얼음까지 남아 있었다.
이 영상: 9,400만 조회.

**[2] 브랜드가 한 일: 없음**
이 영상은 광고가 아니었다.
기획된 것도 아니었다.
실제 사용자의 실제 사고.
→ 브랜드(스탠리)가 만든 콘텐츠가 아니다.

**[3] 왜 이게 광고보다 강한가**
"우리 텀블러는 내구성이 뛰어납니다"
→ 아무도 안 본다.
불탄 차 속 멀쩡한 텀블러
→ 9,400만 명이 봤다.
설명은 무시되고, 증거는 공유된다.

**[4] 숫자로 보는 신뢰 격차**
기업이 만든 콘텐츠 신뢰도: 38%
사용자가 만든 콘텐츠 신뢰도: 70%
(Edelman Trust Barometer, 2024)
→ 같은 사실이라도
   누가 말하느냐가 신뢰를 결정한다.

**[5] CEO의 대응 = 두 번째 바이럴**
스탠리 CEO가 48시간 내 반응:
"새 차를 선물하겠습니다."
→ 이 대응 영상도 수천만 조회.
빠르고 파격적인 대응이 서사를 완성.

**[6] 알고리즘이 증거를 밀어주는 이유**
① 놀라움 = 체류 시간 증가
② 맥락 설명 불필요 = 3초 만에 이해
  → 누구에게나 공유 가능 (언어/문화 무관)
③ 자발적 공유 = 지지가 아닌 발견 보고

**[7] 증거 마케팅의 조건**
⚠️ 텀블러가 녹았으면?
→ 역대급 부정 바이럴.
증거 마케팅의 전제:

## 11. TikTok 시리즈 콘텐츠 — "바이럴 없이 40억 조회"

**[2] 바이럴의 문제**

한 번 터진다 → 다음에 또 터질까?
→ 확률적 사건. 전략이 될 수 없다.

**[3] 플랫폼이 진짜 보는 것**
조회수가 아니다.
세션 시간 (앱 체류 시간)이다.
→ 체류 ↑ → 광고 슬롯 ↑ → 매출 ↑

**[4] 시리즈가 세션을 늘리는 원리**
① 호기심 분할: "다음 편에서 진짜 비밀을..."
   → 정보 갭 이론 (Loewenstein, 1994)
   → 알고 싶은데 아직 모르는 간극 = 중독

② "Part 3 어디?" 댓글 = 내비게이션
   → 댓글이 검색 기능을 대체

**[5] 시리즈 vs 바이럴 데이터**

프로필 방문율: 4배
3편 이상 시청 후 팔로우 전환율: 11.2%
(단일 영상: 3.7%)

**[6] 습관 형성**
"다음 편 나왔을까?" → 앱을 다시 연다.
매몰 비용: 이전 편에 투자한 시간
→ 시리즈를 중단하기 어렵다.
= 넷플릭스 빈지워칭과 같은 구조.

**[7] 플랫폼이 시리즈를 미는 이유**
① 광고 단가: 몰입한 사용자 = 높은 CPM
② 콘텐츠 공급 안정: 정기적 업로드
③ 크리에이터 이탈 방지: 시리즈 중간에
   플랫폼 옮기면 시청자 잃음 → 전환 비용 ↑

**[8] 실무 프레임워크**
① 전체 아크(arc)를 먼저 설계
② 매 편: 독립적 가치 + 다음 편 기대
③ 주 2~3회 정기 업로드
④ 시청 완료율 & 다음 편 전환율 추적

→ 거리 = 자연스러운 보호막.

배달앱: 반경 내 모든 매장이 한 화면에.
→ A브랜드 강남점과 A브랜드 서초점이 나란히.
메뉴 같고, 가격 같고, 프로모션 같다.
= 완벽한 대체재.

**[3] 데이터가 보여주는 것**
349개 매장 분석 결과:
배달 매출 예측에서
"타 브랜드 수"만 넣었을 때 ★★★
"자사 매장 포함"했을 때 ★★★★★
→ 차이 = 자기잠식 효과.

**[4] 거리별 효과**
100m: ★★★★★ (가장 강함)
300m: ★★★★
500m: ★★★
1km: ★★
→ 배달 반경이 겹치면 매출을 나눠 갖는다.

**[5] 매장 식사는?**
매장 식사에서는 자기잠식 효과 = 없음.
직접 가는 사람은 가까운 한 곳만 간다.
→ 자기잠식은 "배달"이라는 채널이
만들어낸 구조적 현상이다.

**[6] 왜 위험한가**
본사: 매장 늘리면 = 가맹비 수입 증가
가맹점주: 배달 반경 겹치면 = 매출 감소
→ 본사의 성장과 가맹점의 수익이
구조적으로 충돌하는 지점.

**[7] 가맹점주는 원인을 모른다**
매출이 줄었을 때:
"경기가 나빠서" "배달비가 올라서"
→ 같은 브랜드 다른 매장이 원인이란 생각은
잘 못 한다. 같은 편이라고 인식하니까.

**[8] 해법**
① 출점 심사 첫 질문:
   "경쟁 브랜드 수"가 아니라
   "우리 매장이 이미 있는가"
② 배달 반경 중복을 네트워크 수준에서 관리
③ 같은 브랜드 내 미세 차별화 실험

**[9] CTA**
배달앱이 공간의 보호막을 제거했다.
같은 브랜드끼리의 경쟁이 시작됐다.

## 14. 새벽배송 효과 — "오프라인 매장이 근처에 있어야 효과 있다"
**Type C (랩 리서치) · 1 캐러셀 · 9장**

**[1] 훅**
새벽배송 후 지출 변화:
근처에 오프라인 매장 없는 고객: 효과 없음
근처에 오프라인 매장 있는 고객: +78%
같은 서비스. 극적인 차이.

**[2] 뭘 분석한 건가**
BDM Lab이 식료품 리테일러
920명 고객의 실제 구매 데이터 분석.
(설문이 아니라 실제 결제 금액.)
이중차분법(DID)으로 인과 추정.

**[3] 전체 결과**
새벽배송 후:
주 평균 지출: 12,000원 → 14,500원
쇼핑 빈도: 2.5주 → 2주
→ 편하면 더 자주, 더 많이 산다.
여기까진 당연하다.

**[4] 진짜 발견**
근처에 매장 없음:
→ 지출 증가 통계적 유의 없음

근처에 매장 있음:
→ 주 +10,556원 (+78%)
→ 쇼핑 빈도 거의 2배

고객 1인당 연 55만 원 추가 매출.

**[5] 왜?**
식료품은 극도의 경험재다.
사진만 보고 수박 당도를 알 수 없다.
→ 오프라인 매장에서 쌓은 품질 신뢰가
온라인 구매의 불확실성을 상쇄한다.

**[6] 더 중요한 발견**
총 식료품 지출은 거의 일정했다.
→ 더 많이 먹은 게 아니다.
경쟁사에 쓰던 돈을 이쪽으로 옮긴 것.
= 월렛 셰어 전환.
새벽배송 = 시장을 키우는 도구가 아니라
경쟁사 고객을 뺏는 무기.

**[7] 실무 시사점**
① 새벽배송 론칭 지역:
   오프라인 매장 커버리지 맵부터 펴라
② 매장 폐점 결정 시:
   그 매장이 지탱하던 온라인 매출도 계산하라
③ "온라인 전환" = 오프라인을 줄이는 것이 아니라
   두 채널 시너지를 설계하는 것

**[8] 쿠팡에게 시사하는 것**
오프라인 없는 플랫폼은
신뢰를 온라인 경험에만 의존.
→ 쿠팡이 오프라인 매장(쿠팡마트)을
여는 건 다각화가 아니라
신뢰 자산 구축 전략일 수 있다.

**[9] CTA**
온라인과 오프라인은
대체 관계가 아니라 보완 관계다.
오프라인이 신뢰를 만들고,
온라인이 편의를 제공한다.

## 15. 커머스 멤버십 — "쿠팡플레이 때문에 가입하는 사람은 없다"
**Type C (랩 리서치) · 1 캐러셀 · 9장**

**[1] 훅**
"로켓와우 가입하면 드라마까지 공짜!"
쿠팡이 쿠팡플레이에 수천억을 투자한 이유.
4,597명의 4년간 카드 결제 데이터를 봤다.
효과 있을까?

**[2] 이 연구의 특징**
설문 아님. 실제 카드 결제 내역.
4,597명 × 4년 (2020~2023)
→ 쿠팡만이 아니라 이마트, 11번가...
전체 소비 포트폴리오를 봤다.

**[3] 5개 가설을 검증했다**
① 총지출이 크면 가입한다 → ✅ 맞다
② 온라인 쇼핑 많이 하면 → ❌ 아니다
③ 쿠팡 지출 비중 높으면 → ✅ 맞다
④ OTT 많이 쓰면 → ❌ 아니다
⑤ 오프라인 활동 많으면 → ❌ 아니다

살아남은 건 딱 2개.

**[4] 발견 1: 절약형이 아니라 소비형**
"배달비 아끼려고 가입"?
→ 데이터는 정반대.
이미 많이 쓰는 사람이 가입한다.
멤버십 = 수요 창출 도구가 아니라
기존 수요를 구조화하는 도구.

**[5] 발견 2: "온라인 쇼핑" ≠ "쿠팡 쇼핑"**
11번가에서 많이 사는 건 무관.
쿠팡에서 많이 사는 게 핵심.
→ 멤버십은 새 고객을 데려오는 도구가 아니라
기존 충성 고객을 고정(lock-in)하는 도구.

**[6] 뼈아픈 결과: OTT 번들링**
넷플릭스에 월 1.5만원 쓰는 사람이
"쿠팡플레이도 공짜니까" 가입? → NO.
OTT 지출과 멤버십 가입은
통계적으로 유의한 관계 없음.
팬데믹 때도, 엔데믹 때도.

**[7] OTT 번들의 역할**
신규 가입 유도 = 실패
기존 가입자 이탈 방지 = 가능성 있음
"해지하면 쿠팡플레이도 못 보는데..."
→ 가입의 동인이 아니라 이탈의 마찰.

**[8] 실무 액션**
① 멤버십 마케팅 타겟:
   "모든 비회원"이 아니라
   "이미 상위 20% 지출 중인 비회원"
② OTT 투자 ROI: 가입이 아닌 이탈률로 재측정
③ 번들 혜택 추가 < 핵심 쇼핑 경험 강화

**[9] CTA**
멤버십 가입을 결정하는 건
번들 혜택이 아니라
이미 그 플랫폼의 핵심 경험에 만족하는 것.

## 16. 위치 데이터 — "카페 수가 매출을 예측한다"
**Type C (랩 리서치) · 1 캐러셀 · 9장**

**[1] 훅**
프랜차이즈 매출 예측 모델을 만들었다.
가장 강력한 변수는
유동인구가 아니었다.
주변 카페 수였다.

**[2] 프로젝트 배경**
프랜차이즈 기업이 BDM Lab에 의뢰.
"신규 매장, 어디에 내야 하는가?"
기존 방식: 부동산 중개인 추천 + 경영진 직관
→ 결과: 20% 이상이 2년 내 폐점.

**[3] 뭘 넣었나**
78개 기존 매장 × 3년 매출 데이터
+ 유동인구 (통신사 데이터)
+ 상권 특성 (경쟁, 임대료, 접근성)
+ 인구통계 (연령, 소득, 주거 유형)
모델: XGBoost + SHAP 분석

**[4] 유동인구의 역설**
유동인구 월 300만: 매출 중위권
유동인구 월 40만: 매출 상위 10%
→ 핵심은 '통과형'이 아니라 '체류형' 유동인구.
체류형의 매출 기여도 = 통과형의 4.2배.

**[5] 카페 밀집도가 왜?**
반경 500m 카페 수 → 전체 예측의 22% 설명.
카페 운영자는 상권 전문가.
카페가 밀집 = 이미 전문가 검증을 통과한 상권.
+ 카페 밀집 = 체류 생태계가 형성된 곳.

**[6] 임대료 함정**
임대료 상위 25%: 매출 +40%
하지만 영업이익률: -3.2%p
좋은 자리에서 더 많이 팔았지만
더 많이 남기진 못했다.
→ 매출 ≠ 수익.

**[7] 한계: 솔직하게**
① 운영 역량은 모델에 안 잡힌다
② 상권은 변한다 (시간적 정태성)
③ 상관 ≠ 인과 (카페 옆 = 성공?)
④ 78개 매장 = 작은 표본
모델은 의사결정 도구이지 의사결정 자체가 아니다.

**[8] 실무 프레임워크**
위치 데이터 = 스크리닝 도구
→ 100개 후보를 10개로 좁히는 데 사용
→ 10개에서 3개는 현장 조사 + 판단
데이터로 명백한 실패를 걸러내되,
성공은 운영으로 만들어라.

**[9] CTA**
위치 데이터는 많은 것을 말해주지만
모든 것을 말해주지 못한다.
데이터 + 판단. 둘 다 필요하다.
