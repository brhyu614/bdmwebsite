9:16 릴스 영상을 빠르게 만드는 도구입니다. 아래 가이드를 따라 시작하세요.

1 Pexels API Key 발급 (무료)

  1. pexels.com/api 에 접속
  2. 계정 생성 (무료) 후 로그인
  3. "Your API Key" 메뉴에서 키 복사
  4. 왼쪽 사이드바 상단 Pexels API Key 입력란에 붙여넣기
API Key는 브라우저에 자동 저장됩니다. 다른 사람과 공유할 필요 없이 각자 발급받아 사용하세요.

2 씬(장면) 구성

릴스는 여러 씬으로 구성됩니다. 각 씬마다:

  • 배경 영상 — Pexels 검색 또는 직접 업로드 (씬마다 다른 영상 가능)
  • 시작/끝 시간 — 해당 씬이 보이는 구간(초)
  • 텍스트 — 제목, 본문, 태그
  • 위치/크기 — 9방향 배치, 3단계 크기

3 텍스트 마크업

4 프리뷰 & 내보내기

  • 재생 버튼으로 전체 릴스 프리뷰
  • 타임라인 클릭으로 원하는 시점으로 이동
  • Export Reels Video로 WebM 영상 다운로드
  • Export Current Frame으로 현재 프레임 PNG 저장
이 파일은 HTML 하나로 동작합니다. 파일만 공유하면 누구나 바로 사용할 수 있고, API Key는 각자 발급받아 사용합니다.
