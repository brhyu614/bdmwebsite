# Threads 프로그램 가이드

> 담당: 사키에
> 프로그램: `program-threads/threads-factory.html`

---

## 프로그램 위치

```
BDM-Assets/
├── program-threads/
│   ├── threads-factory.html        ← 이 파일을 더블클릭해서 브라우저로 열기
│   └── prompts/
│       ├── threads-prompt.txt      ← 생성 프롬프트 (직접 수정 가능)
│       └── review-prompt.txt       ← 자체 점검 프롬프트 (직접 수정 가능)
```

## API 키 입력 (처음 한 번만)

Claude API Key + Gemini API Key (교수님에게 받기)

## 매일 사용법

### 1. 글 선택
- 웹사이트 글 목록에서 선택 또는 직접 입력

### 2. 생성
- "생성하기" 클릭
- AI가 **3개 변형**을 만들어줌 (각각 다른 훅 공식 적용):
  - 반직관 인사이트 ("통념과 달리 사실은...")
  - 놀라운 데이터 포인트 ("X%가...")
  - 프레임 전환 ("사실 이건 Y가 아니라 Z다")
  - 마이크로 케이스 ("[기업명]이 한 일의 결과")
  - 도발적 질문 ("왜 X할까?")

### 3. 결과 확인
- 3개 변형이 Threads 포스트처럼 보여줌
- Gemini가 각 변형의 임팩트/데이터/공유유발 점수를 매기고 "가장 좋은 변형" 추천
- 마음에 드는 변형 클릭하여 선택

### 4. 교수님 피드백
- 교수님에게 3개 변형 보여주기
- "이게 좋은데 좀 더 짧게" 같은 피드백 입력 → 재생성
- 승인

### 5. 내보내기
- "텍스트 복사" → Threads 앱에 직접 붙여넣기
- 이미지가 추천되면 해당 키워드로 이미지 검색해서 첨부

## Threads 콘텐츠 핵심 규칙

1. **300자 이하가 이상적**. 짧을수록 좋음
2. **첫 문장이 전부**. 스크롤 멈추게 하는 한 줄
3. **데이터/숫자 반드시 포함**
4. **마지막에 질문** — 댓글을 유도
5. **해시태그 3-5개만**

## 프롬프트 수정

- "프롬프트 설정" 버튼 → 수정 → 자동 저장
- `program-threads/prompts/threads-prompt.txt` 파일 직접 수정도 가능
- Claude Code로 프로그램 자체도 수정 가능
