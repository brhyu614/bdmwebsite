## 1. 왜 중요한가 — 0.1~0.3초의 법칙

- 클릭 여부는 0.1~0.3초 안에 결정된다. 내용이 아무리 좋아도 클릭이 없으면 소비되지 않는다.
- CTR이 초기 신호이고, 이것이 알고리즘 노출을 눈덩이처럼 확장한다.
- 썸네일·타이틀은 **콘텐츠 기획의 첫 단계**여야 한다. 본문을 다 쓴 뒤 붙이는 장식이 아니다.

## 2. 5가지 코어 원칙

| # | 원칙 | 실행 |
|---|------|------|
| 1 | **시각적 현저성** | 강한 대비, 클로즈업, 화살표. 비슷한 콘텐츠 사이에서 즉시 튀어야 함 |
| 2 | **인지 부하 최소화** | 단어 수 줄이고 구조 단순. 같은 단어 3회 이상 반복 금지 |
| 3 | **감정 메시지** | 편도체 자극. 놀람·의심·배신·공감 중 하나는 건드려야 함 |
| 4 | **호기심 갭 (Curiosity Gap)** | 정보 일부 숨김. "도대체 왜 저래?" 유발이 도파민 클릭으로 직결 |
| 5 | **타겟 확장 언어** | 니치 주제를 더 큰 대중 관심사로 번역. 마케팅 용어 → 일상 언어 |

## 3. 타이틀 구조 공식 (재사용 템플릿)

| 공식 | 템플릿 | 케이스 |
|------|--------|--------|
| **역설/반전** | "~인 줄 알았는데 ~다" / "A는 ~하지만 B는 ~다" | "뒷광고에 지친 시대, 극장에서 돈을 내고 광고를 봤다" |
| **시대 규정 + 행동 모순** | "~ 시대에, ~다" | "광고 신뢰가 무너진 시대에, 티켓이 매진됐다" |
| **직접 질문** | "도대체 왜 ~했을까" | "도대체 왜 사람들은 광고에 1,000원을 냈을까" |
| **숫자 3연타** | "A · B · C: ~" | "1,000원 · 13분 · 칸 5관왕" |
| **리스트형** | "~하는 N가지 방법" | "~없이는 못 산다 시리즈" (GQ Sports) |
| **Before/After** | "-Nkg ~일치" / 변화량 강조 | "CTR 3.7% → 7%로 오르자 조회수 2배" |
| **워드플레이** | 한 글자 대비로 리듬 | "뒷광고" → "앞광고", "협찬" → "광고" |

## 4. 체크리스트 (타이틀 확정 전 필수)

- [ ] **흘긋보기 테스트**: 2초만 보고 뭐가 남나? 한 줄 메시지로 요약되나?
- [ ] **명료성 테스트**: 썸네일이 사이드바 크기로 줄어도 다 보이나?
- [ ] **대비 테스트**: 같은 주제 다른 콘텐츠 사이에서 튀나?
- [ ] **반복 단어 체크**: 같은 단어 3회 이상 나오나? (→ 인지 부하 증가)
- [ ] **호기심 갭 존재?**: 읽고 나서 "왜?" 또는 "어떻게?"가 생기나?
- [ ] **구체성**: 숫자·고유명사·시공간 중 최소 1개가 들어갔나?
- [ ] **AI 티**: "살펴보겠습니다", "다양한", "흥미로운" 같은 상투어 없나?
- [ ] **낚시 체크**: 본문과 불일치하지 않나? (알고리즘 페널티)

## 5. 피해야 할 패턴

- **밋밋한 기술적 제목**: "~에 대한 분석", "~의 이해", "~에 관하여"
- **교과서적 마무리 톤**: "결론적으로 ~이 중요하다"
- **같은 키워드 3회+ 반복**: 예) "광고를 믿지 않는 광고 불신 시대에 광고에 돈을 냈다" → "광고" 3회 → NG
- **낚시 과장**: 내용과 괴리된 자극 → 이탈률↑ → 알고리즘 하락
- **텍스트 과다**: 3단어 이상 연속 수식어는 흘긋 테스트 실패
- **어두운 썸네일**: 밝은 색이 거의 항상 CTR에서 이김 (강의자료 원칙)

## 8. 세션 회고 (케이스에서 배운 것)

### 2026-04-23 — 〈밤낚시〉 세션

**과정 요약:**
1. v0 (Claude 초안): "광고를 돈 내고 극장에서 본다: 〈밤낚시〉가 깬 '스킵의 법칙'" — 설명형, 긴 편
2. 교수님 프레임 지정: "광고 협찬 콘텐츠 신뢰 하락 시대 vs 광고에 돈을 냈다"라는 역설 강조
3. 교수님 제시안: "광고 협찬 콘텐츠를 믿지 않는 광고신뢰 하락 시대에, 광고에 돈을 냈다" — 의미 정확하지만 "광고" 3회 반복으로 인지 부하↑
4. 압축 대안 3종 제시 → 교수님 최종: "뒷광고에 지친 시대, 극장에서 돈을 내고 광고를 봤다"

**세션에서 학습한 패턴:**
- **패턴 A — 돈 흐름 역전 프레임**: "돈 받고 만드는 쪽"과 "돈 내고 소비하는 쪽"의 대칭이 성립하면 강력한 역설. 브랜디드 콘텐츠·인플루언서 관련 주제에 재사용 가능
- **패턴 B — 시대 규정 + 공간/행동 구체성**: "~ 시대에" 설정 후 구체적 공간·행동으로 반전을 실체화하면 추상성이 사라짐. "극장에서 돈을 내고"처럼 공간(극장) + 동사(돈 내다) + 대상(광고)의 3단 구체성 권장
- **교훈 — 한 단어만 집으면 된다**: "협찬·광고·리뷰·PPL" 다 나열할 필요 없음. "뒷광고" 하나만으로 이미 뒷돈 받은 콘텐츠 전체 맥락이 독자 머릿속에서 자동 연상됨

**다음에 바로 적용:**
- 역설 구성 시 "A는 ~하지만 B는 반대~" 대칭 구조를 1차 시도로 설정
- 시대 규정부 단어는 **하나의 강한 단어**로 해결 (여러 개 나열 X)
- 뒤 절은 **공간 + 행동 + 대상** 3단 구체성 확보
