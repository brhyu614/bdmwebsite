## 2. 콘텐츠 믹스

### 주간 게시 목표: 4~5회

| 콘텐츠 유형 | 주간 빈도 | 목적 | 알고리즘 효과 |
|------------|----------|------|-------------|
| **캐러셀** | 3회 | 깊이 있는 인사이트 전달 | 체류시간↑ 저장↑ (가장 높은 인게이지먼트) |
| **릴스** | 1~2회 | 비팔로워 도달 확장 | 도달률 5배↑ (탐색탭 노출) |
| **스토리** | 매일 1~2회 | 팔로워 관계 유지 | 피드 상단 노출 유지 |

### 왜 이 비율인가?
- 2026년 인스타 알고리즘: **캐러셀이 릴스보다 인터랙션 12% 높음**
- 릴스는 비팔로워 도달이 5배 → **새 팔로워 유입**에 필수
- 스토리가 없으면 기존 팔로워 피드에서 밀려남 → **매일 존재감 유지**

| **수요일** | — | 19:00~20:00 |
| **목요일** | 12:00~13:00 | — |
| **토요일** | 11:00~12:00 | 10:00~11:00 |

### 왜 이 시간인가?
- **점심(12~13시)**: 직장인/마케터가 피드를 훑는 시간. 캐러셀은 "읽을거리"이므로 여유 있는 점심에 최적

- **주말 오전(10~12시)**: 여유롭게 스크롤하는 시간. 긴 캐러셀이나 릴스 모두 효과적

## 5. 릴스 전략

### 릴스의 역할
캐러셀이 "본편"이라면, 릴스는 **예고편**이자 **도달 엔진**.

### 릴스 포맷 (30~60초)

**포맷 A: "한 줄 훅 → 반전 → 풀버전은 캐러셀에서"**
```
[0~3초]  텍스트 훅: "치킨집 매출을 결정하는 건 리뷰가 아닙니다"
[3~20초] 핵심 데이터 1~2개 시각화
[20~30초] "전체 분석은 최신 캐러셀에서 → 프로필 링크"
```

**포맷 B: "그거 아셨어요?" 지식 공유형**
```
[0~3초]  "팔로워 10만인데 게시물 도달이 2%인 이유"
[3~45초] 핵심 메커니즘 설명 (자막 + 음성 or 자막 only)
[45~60초] "더 깊은 분석 → 프로필 링크"
```

**포맷 C: "비교" 시리즈**
```
[0~3초]  Split screen 비교 이미지
[3~30초] A vs B 차이점 빠르게
[30~45초] "왜 이런 차이가 나는지 → 캐러셀에서"
```

### 릴스 제작 원칙
1. **처음 3초가 전부** — 3초 유지율 60% 이상이어야 알고리즘이 밀어줌
2. **자막 필수** — 무음 시청 80%+. 자막 없으면 스킵
3. **트렌딩 오디오 활용** — 인스타가 밀어주는 음악 사용
4. **오리지널 콘텐츠** — TikTok 워터마크 절대 금지 (알고리즘 페널티)
5. **Trial Reels 활용** — 비팔로워에게만 먼저 테스트 가능

### 릴스 아이디어 (캐러셀과 1:1 매칭)

| 릴스 | 매칭 캐러셀 | 릴스 훅 |
|------|-----------|---------|
| R1 | 인스타 알고리즘 | "팔로워 10만인데 왜 아무도 안 봐?" |
| R2 | 우는 말 인형 | "공장에서 불량품이 나왔는데 2만개가 팔렸다" |
| R3 | 배민 자기잠식 | "배민에서 당신의 진짜 경쟁자는 옆집 교촌이 아닙니다" |
| R4 | 쿠팡 멤버십 | "쿠팡플레이 때문에 로켓와우 가입한 사람? → 데이터는 아니라고 합니다" |
| R5 | 에이전틱 커머스 | "2년 후, AI가 당신 대신 쇼핑합니다. 진짜로." |
| R6 | 위치 데이터 | "당신의 위치 데이터로 알 수 있는 것 (그리고 알 수 없는 것)" |
| R7 | 치킨집 연구 | "치킨집 349개를 3년간 추적했습니다" |
| R8 | 클로드 슈퍼볼 | "AI 회사가 슈퍼볼 광고에 70억을 쓴 진짜 이유" |

## 6. 스토리 전략

### 목적
- 매일 존재감 유지 (스토리 없으면 피드에서 밀림)
- 팔로워와 양방향 소통
- 캐러셀/릴스 게시 알림

### 스토리 유형 (일주일 루틴)

| 요일 | 스토리 유형 | 예시 |
|------|-----------|------|
| **월** | 이번 주 프리뷰 | "이번 주에 올라올 주제: ____" (기대감) |
| **화** | 캐러셀 알림 + 투표 | 새 캐러셀 올린 후 "이 중에 뭐가 더 놀라웠어요?" 투표 |
| **수** | 릴스 공유 + 질문 | 릴스 게시 후 "비슷한 경험 있나요?" 질문 스티커 |
| **목** | 캐러셀 알림 + 퀴즈 | "치킨집 매출을 가장 크게 좌우하는 건?" 퀴즈 스티커 |
| **금** | 비하인드/랩 일상 | 연구실 일상, 읽고 있는 논문, 데이터 분석 화면 |
| **토** | 주말 콘텐츠 알림 | 새 게시물 + "이번 주 가장 많이 저장된 게시물" 공유 |
| **일** | 가벼운 질문/여담 | "이번 주 마케팅에서 가장 인상 깊었던 뉴스는?" |

### 스토리 필수 요소
- **매번 인터랙션 스티커 1개 이상**: 투표, 퀴즈, 질문, 슬라이더
- 인터랙션 = 알고리즘 신호. "반응할 거리"를 줘야 다음 스토리도 상단에 노출

## 7. 캡션 전략

### 구조
```
[훅 — 첫 문장이 전부. 피드에서 "...더 보기" 전에 보이는 부분]

[본문 — 3~5문장. 핵심 인사이트 요약]

[CTA — 저장/공유/댓글 유도]

[해시태그 — 3~5개]
```

### 캡션 예시 (우는 말 인형)
```
공장에서 입이 거꾸로 붙은 인형이 나왔다.
보통은 폐기한다. 이 회사는 그냥 팔았다.

하루 400개씩 나온 불량품이 20,000개 팔렸다.
정가보다 비싸게.

왜? 사람들이 이 인형을 "발견"했기 때문이다.
알고리즘이 완벽한 제품보다 결함 있는 제품을 더 많이 보여준 구조를 분석했다.

💾 나중에 다시 볼 사람은 저장
📩 마케터 친구에게 공유

#마케팅 #알고리즘 #바이럴마케팅 #BDMLab #빅데이터마케팅
```

### CTA 패턴 (돌려가며 사용)
1. **저장 유도**: "💾 나중에 다시 보려면 저장" / "이 데이터 기억하고 싶으면 저장해두세요"
2. **공유 유도**: "📩 마케터 친구한테 보내주세요" / "이 분석이 필요한 사람 태그"
3. **댓글 유도**: "가장 놀라운 숫자는 몇 번? 댓글로" / "여러분의 경험은 어떤가요?"
4. **웹 유도**: "전체 분석 + 출처 → 프로필 링크"

### 해시태그 전략 (3~5개만)
2026년 인스타 알고리즘에서 해시태그는 "분류 라벨" 역할. 많이 쓰면 오히려 도달 감소.

**고정 태그 (매번)**:
- `#BDMLab` (브랜드)
- `#빅데이터마케팅` (니치)

**회전 태그 (주제별 1~3개)**:
- 알고리즘 계열: `#알고리즘` `#마케팅전략` `#소셜미디어마케팅`
- 리서치 계열: `#데이터분석` `#마케팅리서치` `#소비자행동`
- AI 계열: `#AI마케팅` `#AEO` `#에이전틱커머스`
- 브랜드 계열: `#바이럴마케팅` `#브랜드전략` `#디지털마케팅`

- 타겟 팔로워가 모인 계정에서 의미 있는 댓글 5~10개
- 관련 해시태그 탐색 → 좋은 콘텐츠에 공감 댓글

### 타겟 인게이지먼트 계정 (댓글 활동)
- 마케팅 관련 인스타 계정
- 비즈니스/스타트업 커뮤니티 계정
- 대학원/연구 관련 계정
- 데이터/AI 관련 교육 계정

### 핵심 원칙
- **의미 있는 댓글만**: "좋아요!" 한 줄이 아니라 구체적 의견 (2~3문장)
- "이 데이터 흥미롭네요. 저희 랩에서도 비슷한 결과를 봤는데..." 식으로 전문성 어필
- **팔로우 구걸 금지**: "맞팔해요" 절대 안 함. 콘텐츠로만 유입

## 13. Phase 2 이후 (기존 캐러셀 소진 후)

### 콘텐츠 파이프라인
```
새 웹 아티클 발행 (주 1~2회)
    ↓
캐러셀 제작 (아티클 핵심 요약)
    ↓
릴스 제작 (30초 훅 버전)
    ↓
인스타 게시 (화·목 캐러셀 / 수 릴스)
    ↓
스토리로 알림 + 인터랙션
```

### 추가 콘텐츠 아이디어
1. **"1분 데이터"** 릴스 시리즈 — 매주 하나의 데이터 포인트를 60초로
2. **팔로워 Q&A** — 질문 모아서 스토리 or 캐러셀로 답변
3. **"이번 주의 숫자"** — 매주 월요일 인상적인 마케팅 데이터 1개
4. **콜라보** — 마케팅 실무자/다른 교수 계정과 공동 라이브 or 캐러셀
5. **리서치 티저** — 진행 중인 연구의 중간 발견 공유
