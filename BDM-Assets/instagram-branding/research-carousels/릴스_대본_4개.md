## 릴스 1: 스탠리 텀블러 — "증거가 설명을 이긴다"

**길이:** 45초
**톤:** 차분하지만 확신 있게
**포맷:** 텍스트 오버레이 + 실제 영상 클립 편집

### 대본

```
[0-5초] 훅 — 화면에 큰 텍스트
"이 영상 하나로
스탠리 텀블러가 완판됐습니다."

[5-12초] 상황 설명
불에 탄 차 안에서 멀쩡한 텀블러 영상 (틱톡 원본 캡처 or 재현).
"2023년 11월, 틱톡에 올라온 영상 하나.
차가 전소됐는데, 컵홀더의 스탠리 텀블러만 멀쩡합니다.
얼음도 안 녹았습니다."

[12-20초] 결과
"이 영상, 9,400만 뷰.
스탠리는 전 매장 품절.
CEO가 직접 댓글 달고, 차를 새로 사줬습니다."

[20-32초] 왜?
"스탠리가 '내구성이 뛰어납니다'라고 광고하면?
아무도 안 믿습니다.
기업 콘텐츠 신뢰도 38%.

그런데 실제 사고 현장에서 살아남은 증거?
UGC 신뢰도 70%.

설명은 무시당하고, 증거는 공유됩니다."

[32-40초] 핵심 한 줄
텍스트 강조:
"증거 > 설명.
이게 알고리즘 시대의 마케팅 법칙입니다."

[40-45초] CTA
"이런 분석, 프로필 링크에서 더 봅니다.
BDM Lab."
```

### 편집 지시

- 배경: 다크 톤
- 텍스트: 흰색 + 핵심 수치는 그린(#059669) 또는 노랑 강조
- 틱톡 원본 영상은 저작권 이슈 → 캡처 이미지 + "출처: TikTok @danimarielettering" 표기, 또는 유사 상황 재현
- BGM: 긴장감 있는 미니멀 비트 (저작권 프리)
- 마지막에 BDM Lab 로고 + "LAB RESEARCH" 라벨

## 릴스 2: 이케아 펀치 원숭이 — "0원 마케팅의 법칙"

**길이:** 50초
**톤:** 스토리텔링, 약간 감성적
**포맷:** 교수님 직접 출연 (상반신) + B-roll 편집

### 대본

```
[0-5초] 훅
"이케아가 광고비 0원으로
인형을 완판시킨 방법."

[5-15초] 스토리
"호주의 한 동물원에서 태어난 아기 원숭이.
어미가 거부했습니다.
사육사가 대신 안아줄 인형을 골랐는데,
그게 이케아의 19달러짜리 오랑우탄 인형이었습니다."

[15-25초] 바이럴
"펀치가 인형을 안고 있는 사진이 퍼지자,
이케아 호주 판매량 200% 증가.
이베이에서 정가 19달러 인형이
최고 350달러에 거래됩니다.
17.5배."

[25-35초] 왜?
"여기서 핵심은 '신뢰 비용'입니다.
사육사가 이 인형을 고른 건
이케아 광고 때문이 아닙니다.
아기 원숭이한테 안전한 인형이 필요했을 뿐.
상업적 의도가 제로.
그래서 신뢰가 최대."

[35-45초] 법칙
텍스트 강조:
"연출된 진정성은 없습니다.
신뢰 비용이 0일 때,
전환은 최대가 됩니다."

[45-50초] CTA
"전체 분석은 프로필 링크.
BDM Lab."
```

### 편집 지시

- 펀치 원숭이 사진/영상: Symbio Wildlife Park 공식 소셜에서 인용 (출처 표기)
- 이케아 DJUNGELSKOG 인형 사진
- 가격 변화 그래프: $19.99 → $350 (텍스트 애니메이션)
- BGM: 감성적 피아노 → 전환 시 비트 드롭
- 톤: 기존 알고리즘 디코드 릴스와 동일한 밝은(?) 톤이 아니라, 좀 더 차분하고 학술적

## 릴스 3: 쿠팡 로켓와우 — "쿠팡플레이 때문에 가입한 사람은 없다"

**길이:** 40초
**톤:** 단호하고 직설적
**포맷:** 텍스트 중심 편집 (빠른 컷)

### 대본

```
[0-4초] 훅 — 큰 텍스트
"쿠팡플레이 때문에
로켓와우에 가입한 사람은 없습니다."

[4-10초] 맥락
"쿠팡은 로켓와우에 OTT를 끼워 넣었습니다.
네이버는 시리즈온을 번들링했습니다.
'영상까지 공짜니까 가입하겠지?'"

[10-18초] 데이터
"4,597명의 4년간 카드 결제 데이터.
설문이 아니라, 실제로 어디에 얼마를 썼는지.
5개 가설을 세웠습니다.
2개만 살아남았습니다."

[18-28초] 결과 (빠르게)
텍스트 순서대로:
"✕ OTT 많이 보는 사람 → 가입 안 함
✕ 온라인 쇼핑 비중 높은 사람 → 상관없음
✕ 오프라인 레저 많은 사람 → 상관없음

✓ 이미 쿠팡에서 많이 쓰는 사람
✓ 전체 소비가 큰 사람"

[28-36초] 핵심
"멤버십은 새 고객을 데려오는 도구가 아닙니다.
이미 많이 쓰는 고객을 묶는 도구입니다.
OTT 번들링에 마케팅 비용 쓰지 마세요."

[36-40초] CTA
"전체 분석 → 프로필 링크.
BDM Lab."
```

### 편집 지시

- 배경: 순수 블랙
- 텍스트: 큰 글씨, 한 번에 한 문장씩
- ✕ 는 빨간색, ✓ 는 그린으로
- 쿠팡 앱 캡처 화면 삽입 (로켓와우 가입 화면)
- BGM: 미니멀 일렉트로닉, 빠른 템포
- 컷 전환: 1.5-2초 간격으로 빠르게

## 릴스 4: NPC 스트리머 — "관객을 조종자로 바꾸는 구조"

**길이:** 45초
**톤:** 흥미로운 발견을 공유하는 느낌
**포맷:** 교수님 출연 + NPC 스트리밍 클립 편집

### 대본

```
[0-5초] 훅
"왜 NPC 스트리머한테
돈을 던지는 걸까요?"

[5-12초] 현상
"핑키돌. 틱톡 라이브에서
'아이스크림~' '갱갱~' 반복하면서
수십억을 법니다.
바보 같아 보이죠?"

[12-20초] 구조
"그런데 이건 정교한 설계입니다.
일반 라이브: 당신은 관객.
평균 체류 시간 1.7초.

NPC 라이브: 당신은 조종자.
선물을 보내면 → 반응이 나옵니다.
안 보내면 → 아무 일도 안 일어납니다."

[20-30초] 메커니즘
"심리학에서 이걸
'가변 보상(Variable Reward)'이라고 합니다.
슬롯머신이 같은 원리.
보상이 예측 불가능할 때
사람은 멈추지 못합니다.

NPC 스트리머가 선물마다
다른 반응을 보이는 건
우연이 아니라 설계입니다."

[30-40초] 핵심
"플랫폼 알고리즘은
체류 시간이 긴 콘텐츠를 밀어줍니다.
NPC 라이브는 관객을 참여자로 바꿔서
체류 시간을 극대화합니다.
바보같은 콘텐츠가 아니라,
가장 똑똑한 알고리즘 해킹입니다."

[40-45초] CTA
"이런 분석 더 보려면
프로필 링크. BDM Lab."
```

### 편집 지시

- NPC 스트리밍 클립: 틱톡/유튜브에서 핑키돌 or 유사 스트리머 클립 인용 (출처 표기)
- "관객 → 조종자" 전환을 시각적으로 표현 (텍스트 애니메이션)
- 슬롯머신 GIF/영상 삽입 (가변 보상 설명 시)
- BGM: 게임 느낌의 전자음
- 핑키돌 반응 컷을 빠르게 3-4개 이어붙이기
