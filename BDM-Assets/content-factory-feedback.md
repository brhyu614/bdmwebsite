## 작성 방법

아래 양식대로 적어주세요. 간단하게 적으면 됩니다.

```
### [날짜] [이름]
- 종류: 버그 / 불편 / 아이디어
- 내용: 무엇이 문제인지, 무엇을 원하는지
- (선택) 스크린샷 파일명: screenshot_날짜_설명.png
```
