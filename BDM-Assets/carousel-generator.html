Carousel Generator 사용 가이드
텍스트만 입력하면 배경 영상이 자동으로 삽입된 캐러셀을 만들 수 있습니다.
1 Pexels API Key 발급
pexels.com/api 에서 무료 가입 후 API Key를 발급받으세요.
발급받은 키를 왼쪽 상단 입력란에 붙여넣기하면 됩니다.
API Key는 각자 발급받아 사용합니다. (무료, 1분 소요)
2 슬라이드 구성
왼쪽 패널에서 슬라이드를 추가/삭제/선택할 수 있습니다.
각 슬라이드마다 Tag(카테고리), Title(제목), Body(본문)을 입력하세요.
배경 영상은 텍스트 기반으로 자동 검색되며, 키워드를 직접 지정할 수도 있습니다.
3 텍스트 마크업 문법
**텍스트** → 볼드(강조)
{{텍스트}} → 형광 초록 하이라이트
줄바꿈은 입력창에서 Enter를 누르면 그대로 반영됩니다.
4 내보내기 (Export)
Export All Video — 전체 슬라이드를 WebM 영상으로 다운로드
Export Current Video — 현재 슬라이드만 영상으로 다운로드
Export Current PNG — 현재 슬라이드를 이미지로 다운로드
Duration에서 각 영상의 길이(초)를 설정할 수 있습니다.
Tip: Variant(배경 스타일), Decoration(격자/도트), Gradient Overlay(그라데이션)를 조합하면 다양한 분위기를 연출할 수 있습니다. 키보드 ← → 키로 슬라이드를 빠르게 전환할 수 있습니다.
